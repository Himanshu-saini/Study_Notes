===========
Controller
===========

- logic/backend code function comes in controller file 
- We can have multiple controllers

- Example Controller:
  --------------------
  class <controller_name> < ApplicationController
	def index 
	end

	def page2
    end
  end 

- Instance Variable
  -----------------
  - We can use instance vairable in controller to send data
    the corresponding views
  - Ex.
    Comntrollers/home_controller.rb
	------------
	def home
	  @userName = current_user.name
	end
	
	home/index.html.erb
	------------
	<h1> <%= @userName %> <h1>

- Create Cnntrollers
  -----------------
  - $ rails generate controller <controllerName> <actionName> --skip-routes
    - controllerName : fileName/className of controller
    - actionName : name of function/route
	- --skip-routed : not create a seperate route for this controller 
	- Create 
	  - controller file 
	  - views file 
	  - scss file 
	  - route 
	  - test files
