ECMAscript (ES)
===============
- ECMA - European Computer Manufacturing Association
- ECMAScript is a general-purpose programming language, 
  standardized by Ecma International 
- It is a JavaScript standard meant to ensure 
  the interoperability of Web pages across different Web browsers.
 
- Languages following it:
  - JavaScript, ActionScript, Nashorn.

- After ES%/ES6 versions javascript become much powerful
