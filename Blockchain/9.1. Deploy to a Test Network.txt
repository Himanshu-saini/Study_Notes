INFURA:
=========
It is used to get an etherium node.
To get a node create account on INFURA.IO 
Switch to test network in metamask and add account to it.
Add ethers to that account. This account will be used to deploy the contract.
Go to remix.etherium.
Create a new file.
Copy contrat to it.
Select the test network in option.
Deploy the contract to the network.
Copy the node address where the contract is deployed and transaction number.
Set new network settings in config file with the provided node address and transaction number.
