Amazon Relational Database Services (RDS)
========================================
- RDS Available
  - Amazon Aurora
  - MySQL
  - MariaDB
  - Oracle
  - Postgre
  - Microsoft SQL server

- Amazon Aurora 
  -----------
  - Pricing is 1/10th of enterprise RDS 
  - MySQL or PostgreSQL conpatible
  - 5X faster than MySQL 
  - 3X faster than standard PostgreSQL
  - 15 low-latency read replicas

Amazon NoSQL 
=============
- Amazon DynamoDB
  ---------------
  - Fully managed NoSQL services
  - low latency queries
  - Fine grained access control
  - Regional and Global options

Databse Migration Service 
==========================
- Migrate database to AWS quickly ans securely
- Homogenoeus Migration 
  - Migrate to same database 
- Hetrogenous Migration 
  - https://aws.amazon.com/dms/schema-conversion-tool/
  - Migrate to different type of database 
  - SCT : Schema Convertion tool
    - Convrt schema during hetrogeneous migration
