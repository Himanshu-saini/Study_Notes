========================
Font typefaces in latex
========================

- change the fond typeface of the entire document
  - \usepackage{tgbonum}
  - establishes the font family whose font package name is tgbonum, 
    as the default font for this document

Changing default font typeface
--------------------------------
- \fontfamily{qcr}\selectfont
- set the TEX gyre cursor font typeface, whose fontcode is qcr
  for the text inside the braces

- Ex.
  {\fontfamily{qcr}\selectfont
  This text uses a different font typeface
  }
