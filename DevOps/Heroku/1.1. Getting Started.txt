================
Getting Started
================

- Create Account on Heroku
- Select language to be run here

Linux
------
- Install heroku-cli
  - $ sudo snap install --classic heroku

- Login into heroku from terminal
  - $ heroku login
    - Enter email, password

- Interectave Shell
  - $ heroku run bash
- Any commands after the heroku run are executed on a Heroku dyno

View logs
---------
- $ heroku logs
- $ heroku logs --tail
  - full stream of logs
